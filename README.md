# Python Mini Projects

A collection of small Python projects and automation scripts built to practice real-world programming concepts.

These projects focus on:
- File handling
- Automation
- Web requests and scraping
- Generating documents
- Working with external libraries

---

## Projects

### Quiz Generator
Generates multiple-choice quizzes about world capitals and exports them to TXT, PDF, or DOCX.

Location:
quiz_world/

---

### Link Verification Tool
Scans a webpage and detects broken links (404 errors).

Location:
link_verifier/

---

## Requirements

Some projects may require additional libraries.  
Each project folder contains its own requirements.txt file.

To install dependencies for a specific project:

    python -m pip install -r requirements.txt

---

## Purpose

This repository is intended for practice and experimentation while learning Python and building small, useful tools.
